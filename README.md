# Rock Paper Scissors Game

A classic, interactive, and visually engaging Rock-Paper-Scissors game built with modern web technologies. Play against the computer and track your score in real-time!

## 🚀 Overview

This project is a web-based implementation of the traditional Rock-Paper-Scissors game. It provides a simple yet fun user experience where players can test their luck against a randomly generated computer move.

### 🎯 Why this project?

- **Learning Tool**: Built to demonstrate core web development concepts like DOM manipulation, event handling, and basic game logic.
- **User Interface**: Focuses on creating a clean, responsive, and intuitive UI using CSS Flexbox and modern design principles.
- **Logic Practice**: Features a simple "AI" for the computer's moves using JavaScript's random number generation.

---

## 🎮 How it Works

The game follows the standard rules:
1. **Rock** beats **Scissors**.
2. **Scissors** beats **Paper**.
3. **Paper** beats **Rock**.

### Steps to Play:
- Click on one of the three images (**Rock**, **Paper**, or **Scissors**).
- The computer will simultaneously make its choice.
- The game compares the choices and declares a winner or a draw.
- The message area updates with the result, changing color (Green for Win, Red for Loss) to provide immediate feedback.
- The scoreboard keeps track of the total wins for both you and the computer.

---

## 🛠️ Tech Stack

- **HTML5**: For structured content and semantic markup.
- **CSS3**: For styling, layout (Flexbox), and interactive hover effects.
- **JavaScript (ES6)**: For game logic, random move generation, and dynamic UI updates.




## 📂 Project Structure

- `index.html`: The core structure of the game.
- `style.css`: All visual styling and layout.
- `app.js`: The engine that runs the game logic.
- `*.png`: Assets for Rock, Paper, and Scissors.
